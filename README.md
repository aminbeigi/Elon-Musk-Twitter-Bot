# ElonMuskTwitterBot
## Description
A Java-based program that uploads images (ElonMeme.png or ElonSmoke.png) replying to any new tweets by the twitter account @elonmusk, will refresh the page every minute to check for new tweets.
## Requirements
- Java
- Selenium
- Chromedriver
- A Twitter account
